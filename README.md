# inRECORD

Where Independence Sounds Infinite

A next-generation independent record label blending artistry, technology, and community.

## Features

- **Academy**: Artist development through mentorship, production labs, and business literacy
- **Studio**: Creative HQ in Montréal with Control Room, Sound Lab, AI Suite, and Immersive Listening Room
- **AI Lab**: Exploring adaptive mastering and generative soundscapes
- **DAO**: Fan-powered governance for artwork selection, project funding, and exclusive access

## Tech Stack

- Next.js 14
- TypeScript
- Tailwind CSS
- React 18

## Getting Started

First, install dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Deployment

This project is deployed on [Vercel](https://vercel.com/).
